How to access network shares

Roms and configs are stored on your Retropie/Recalbox installation, and are accessible through network shares. This app allows to directly write on these shares.

Windows

On Windows, you can access the shares very simply:

  • Roms: \\retropie\roms or \\recalbox\\roms
  • Configs: \\retropie\configs or \\recalbox\share\system\configs

MacOS

On MacOS, first you have to mount the Samba shares.

  1. Open a Finder window, then select menu Go > Connect to server
  2. Enter: smb://retropie/ or smb://recalbox/
  3. When asked for a password, connect as guest
  4. Select the folder(s) you wish to mount

Once the folder is mounted, you can access through /Volumes/name_of_share

Linux

Mounting a network share depends on your Linux distribution, and tends to change from version to version. Using the "browse" buttons is probably the easiest way.
